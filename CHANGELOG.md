# Історія змін YouTube Music Proxy

## [1.0.0] - 2024-02-11

### Додано
- Базова функціональність додатку
- Відтворення аудіо з YouTube без реклами
- Пошук музики, виконавців, плейлистів
- Створення та управління плейлистами
- Улюблені треки
- Завантаження аудіо для офлайн прослуховування
- Персоналізовані рекомендації
- Синхронізація з YouTube акаунтом
- Темна тема інтерфейсу
- Гарячі клавіші
- Налаштування якості аудіо
- Автоматичне кешування
- Сповіщення про статус завантажень
- Мінімізація в системний трей

### Технічні деталі
- Backend: Python + FastAPI
- Frontend: Tauri (Rust + Vue.js)
- База даних: SQLite
- Аудіо обробка: yt-dlp + ffmpeg
- Кешування: Локальна файлова система
- Платформа: CachyOS/Linux

## [0.9.0] - 2024-02-05

### Додано

## Політика версій

Ми дотримуємось семантичного версіонування (SemVer):

- **MAJOR** версія: Зміни, що ламають зворотню сумісність
- **MINOR** версія: Додавання нової функціональності з зворотною сумісністю
- **PATCH** версія: Виправлення помилок з зворотною сумісністю

## Відомі проблеми поточної версії

### Продуктивність
- Високе використання пам'яті при великій кількості плейлистів
- Повільне завантаження великих плейлистів
- Затримки при пошуку в офлайн режимі

### UI/UX
- Відсутність анімацій переходів
- Обмежена кастомізація інтерфейсу
- Відсутність режиму високої контрастності

### Функціональність
- Обмежена підтримка міжнародних кодувань
- Відсутність підтримки прямого стрімінгу
- Обмежена фільтрація в пошуку
