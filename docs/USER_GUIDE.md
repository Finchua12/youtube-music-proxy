# Посібник користувача YouTube Music Proxy

## Вступ

YouTube Music Proxy - це десктоп додаток для CachyOS, який дозволяє прослуховувати музику з YouTube без реклами. Додаток забезпечує високоякісний аудіо стрімінг, створення плейлистів, персоналізовані рекомендації та інтуїтивно зрозумілий інтерфейс.

## Встановлення

Для встановлення додатку дотримуйтесь інструкцій у файлі [SETUP.md](SETUP.md).

## Запуск додатку

### Перший запуск

1. Запустіть додаток двічі клікнувши на іконку або з терміналу:
   ```bash
   ./youtube-music-proxy
   ```

2. При першому запуску ви побачите вітальний екран з основними функціями додатку.

3. Додаток автоматично створить необхідні каталоги для зберігання даних:
   - Кеш аудіо: `~/.cache/youtube-music/`
   - База даних: `~/.local/share/youtube-music/database.db`
   - Налаштування: `~/.config/youtube-music/`

## Основний інтерфейс

### Панель навігації

Ліва бічна панель містить основні розділи:
- **Головна**: Персоналізована стрічка з рекомендаціями
- **Пошук**: Пошук музики, виконавців, плейлистів
- **Бібліотека**: Ваші плейлисти, улюблені треки, завантаження
- **Плейлисти**: Швидкий доступ до створених плейлистів

### Верхня панель

- Логотип та назва додатку
- Поле пошуку
- Кнопки керування вікном (згортання, максимізація, закриття)
- Кнопка налаштувань користувача

### Плеєр

Нижня панель плеєра завжди видима і містить:
- Інформацію про поточний трек (обкладинка, назва, виконавець)
- Кнопки керування (попередній, відтворення/пауза, наступний)
- Полосу прогресу
- Кнопки гучності
- Кнопки додаткових функцій (повтор, перемішування, черга)

## Пошук музики

### Використання пошуку

1. Клікніть на поле пошуку в верхній частині вікна або перейдіть до розділу "Пошук"
2. Введіть назву пісні, виконавця або альбом
3. Натисніть Enter або клікніть на іконку пошуку

### Фільтри пошуку

Після пошуку ви можете фільтрувати результати:
- **Все**: Всі типи контенту
- **Треки**: Лише аудіо треки
- **Виконавці**: Музичні гурти та солісти
- **Плейлисти**: Колекції треків

### Взаємодія з результатами

- **Клік на трек**: Почати відтворення
- **Подвійний клік**: Додати до черги
- **Правий клік**: Відкрити контекстне меню (додати до плейлиста, улюблене, завантажити)

## Відтворення музики

### Керування плеєром

- **Відтворення/Пауза**: Пробіл або клік на кнопку
- **Наступний трек**: Ctrl+→ або клік на кнопку
- **Попередній трек**: Ctrl+← або клік на кнопку
- **Перемотка**: Клік на полосу прогресу
- **Гучність**: Колесо миші на плеєрі або переміщення повзунка

### Черга відтворення

- Клік на іконку черги в плеєрі, щоб відкрити список
- Перетягування треків для зміни порядку
- Правий клік для видалення з черги

### Режими відтворення

- **Повтор**: Повторення поточного треку або всієї черги
- **Перемішування**: Випадковий порядок відтворення
- **Один трек**: Автоматична зупинка після закінчення поточного треку

## Бібліотека

### Улюблені треки

- **Додавання**: Клік на сердечко біля треку або через контекстне меню
- **Перегляд**: Розділ "Улюблені" у бібліотеці
- **Видалення**: Клік на забарвлене сердечко або через контекстне меню

### Завантажені треки

- **Завантаження**: Правий клік на треку → "Завантажити"
- **Перегляд**: Розділ "Завантажені" у бібліотеці
- **Видалення**: Правий клік на завантаженому треку → "Видалити"

### Нещодавно додані

- Автоматично оновлюється список останніх 50 треків
- Доступний у розділі "Нещодавні" у бібліотеці

## Плейлисти

### Створення плейлиста

1. Перейдіть до розділу "Бібліотека"
2. Клікніть на кнопку "Створити плейлист"
3. Введіть назву плейлиста
4. Клікніть "Створити"

### Додавання треків до плейлиста

- **Через пошук**: Правий клік на треку → "Додати до плейлиста" → Виберіть плейлист
- **Через плейлист**: Відкрийте плейлист → Клік на "+" → Пошук треків

### Управління плейлистом

- **Перейменування**: Правий клік на плейлисті → "Перейменувати"
- **Видалення**: Правий клік на плейлисті → "Видалити"
- **Експорт**: Правий клік на плейлисті → "Експортувати"
- **Імпорт**: Кнопка "Імпортувати плейлист" у бібліотеці

### Взаємодія з плейлистом

- **Відтворення**: Клік на кнопку відтворення на обкладинці плейлиста
- **Перемішування**: Клік на кнопку перемішування
- **Сортування**: Клік на заголовки колонок (номер, назва, тривалість)

## Рекомендації

- Оновлюються на основі історії прослуховування
- Розділені на категорії: "Для вас", "Популярне", "Відкрити для себе"

## Налаштування

### Налаштування акаунта

1. Клікніть на аватар користувача в правому верхньому куті
2. Виберіть "Налаштування акаунта"
3. Ви можете:
   - Під'єднати YouTube акаунт для синхронізації
   - Управляти улюбленими плейлистами з YouTube
   - Налаштувати приватність

### Налаштування додатку

1. Клікніть на іконку налаштувань (шестерня) в лівому нижньому куті
2. Вкладки налаштувань:

   **Загальні:**
   - Мова інтерфейсу
   - Тема (світла/темна)
   - Автозапуск при вході в систему

   **Аудіо:**
   - Якість аудіо (96k, 128k, 192k, 320k)
   - Вихідне аудіо пристрій
   - Нормалізація гучності

   **Завантаження:**
   - Директорія завантажень
   - Максимальний розмір кешу
   - Автоматичне очищення

   **Проксі:**
   - Налаштування проксі-сервера (якщо необхідно)

   **Сповіщення:**
   - Сповіщення про нові рекомендації
   - Сповіщення про завершення завантажень
   - Сповіщення про помилки

### Гарячі клавіші

- **Пробіл**: Відтворення/Пауза
- **Ctrl+→**: Наступний трек
- **Ctrl+←**: Попередній трек
- **Ctrl+↑/↓**: Регулювання гучності
- **Ctrl+F**: Фокус на пошук
- **Ctrl+L**: Фокус на бібліотеку
- **Ctrl+S**: Зберегти поточний трек до улюблених
- **Ctrl+D**: Завантажити поточний трек
- **Ctrl+Q**: Додати до черги
- **Esc**: Закрити відкрите меню або вікно

## Синхронізація з YouTube

### Підключення акаунта

1. Перейдіть до налаштувань акаунта
2. Клікніть "Під'єднати YouTube акаунт"
3. Увійдіть у свій Google акаунт
4. Надайте необхідні дозволи

### Синхронізовані дані

- Улюблені треки та плейлисти
- Історія прослуховування
- Підписки на канали
- Особисті плейлисти (якщо публічні)

### Керування синхронізацією

- **Автоматична синхронізація**: Щодня о 03:00
- **Ручна синхронізація**: Кнопка "Синхронізувати зараз" у налаштуваннях
- **Відключення**: Кнопка "Від'єднати акаунт"

## Усунення несправностей

### Поширені проблеми

#### 1. Треки не відтворюються

**Перевірте:**
- Наявність інтернет-з'єднання
- Доступність YouTube
- Налаштування проксі (якщо використовується)
- Наявність необхідних кодеків

#### 2. Повільне завантаження

**Рішення:**
- Перевірте швидкість інтернету
- Зменшіть якість аудіо в налаштуваннях
- Очистіть кеш у налаштуваннях

#### 3. Проблеми з пошуком

**Рішення:**
- Перевірте правильність написання запиту
- Спробуйте інші ключові слова
- Перевірте налаштування проксі

#### 4. Додаток не запускається

**Рішення:**
- Перевірте системні вимоги
- Перевстановіть додаток
- Перевірте логи у `~/.local/share/youtube-music/logs/`

### Логи та діагностика

Додаток зберігає логи для діагностики проблем:
- **Backend логи**: `~/.local/share/youtube-music/logs/backend.log`
- **Frontend логи**: `~/.local/share/youtube-music/logs/frontend.log`

Для отримання додаткових логів запустіть додаток з терміналу:
```bash
./youtube-music-proxy --verbose
```
